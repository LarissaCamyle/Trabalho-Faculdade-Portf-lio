# 💻 Portfólio - Larissa Camyle

Este projeto é o meu portfólio pessoal desenvolvido para apresentar minhas habilidades, formação, experiências e projetos na área de tecnologia e desenvolvimento de software.

O site reúne informações sobre minha trajetória acadêmica, conhecimentos técnicos, projetos desenvolvidos e formas de contato, funcionando como uma vitrine do meu trabalho e aprendizado na área de desenvolvimento.

## 🚀 Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  
- SCSS  
- Figma (prototipação e design)

## 📚 Sobre o projeto

O portfólio foi desenvolvido com o objetivo de organizar e apresentar meus principais projetos e experiências na área de tecnologia.  

Nele é possível encontrar:

- Informações pessoais e habilidades interpessoais  
- Formação acadêmica e certificados  
- Conhecimentos em desenvolvimento Front-End e Back-End  
- Experiência profissional  
- Projetos desenvolvidos durante minha jornada de estudos  
- Formas de contato

O layout foi pensado para ser visualmente organizado, intuitivo e responsivo, proporcionando uma boa experiência para quem visita o site.

## 🧠 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei conceitos importantes como:

- Estruturação de páginas com HTML
- Estilização e organização visual com CSS e SCSS
- Boas práticas de organização de código
- Estruturação de um portfólio profissional
- Planejamento de layout e experiência do usuário
