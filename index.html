Sistema de Controle de Estoque

O Sistema de Gerenciamento de Estoque foi desenvolvido em Python com interface via terminal e utilização de planilhas Excel como banco de dados. A aplicação permite registrar requisições de materiais, cadastrar compras, atualizar status de validação e entrega, além de adicionar novos itens ao estoque. O sistema possui validação de entradas para evitar erros no preenchimento e realiza tarefas automaticamente.
